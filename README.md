# Random Colour Pallette Generator

## Overview

A randomly generated colour pallette for web developers that also outputs the RGB, CMYK, and HSB values of the colours it generates.

This utilizes two APIs, Colormind.io and thecolourapi.com to generate a colour pallette of five colours. These are generated every time the page loads. This also has a button for web developers that copies the five colours' hex codes and formats them to place into the root of a css file.
